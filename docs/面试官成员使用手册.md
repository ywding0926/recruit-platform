# MachinePulse 招聘管理系统 — 面试官 / 成员使用手册

---

## 一、系统简介

MachinePulse 招聘管理系统是公司内部的招聘协作平台。作为面试官（成员角色），您的主要职责是：

- 查看分配给您的候选人信息
- 按时参加面试
- 提交面试评价

本手册将指导您完成上述操作。

---

## 二、登录系统

### 2.1 飞书扫码登录（推荐）

1. 打开系统地址（由 HR 提供）
2. 点击页面上的 **"飞书登录"** 按钮
3. 使用飞书 App 扫描二维码
4. 在手机上确认授权
5. 自动跳转至系统首页

> 首次登录时，系统会自动将您的飞书账号与系统中的成员信息关联。

### 2.2 账号密码登录

如果 HR 为您创建了账号密码，也可以通过用户名+密码方式登录。

---

## 三、首页仪表盘

登录后首先看到仪表盘页面，包含：

### 3.1 概览数据
- 候选人总数、进行中数量、本月新增、在招职位数等统计卡片

### 3.2 待办提醒（重点关注）

#### 今日面试
- 显示您今天需要参加的面试列表
- 信息包含：候选人姓名、面试轮次、面试时间
- **点击候选人姓名** 可进入详情页查看候选人资料

#### 待面评
- 显示您已完成面试但 **尚未提交评价** 的候选人
- 请及时提交面评，以免影响招聘流程推进
- 点击可直接进入评价页面

---

## 四、查看候选人

### 4.1 候选人列表

路径：顶部导航 → **候选人**

- 成员角色只能看到与自己 **分配岗位相关** 的候选人
- 支持按姓名、手机号、备注关键词搜索（输入即搜，无需点击按钮）
- 支持按岗位和来源筛选

### 4.2 候选人详情页

点击候选人姓名进入详情页，可查看：

| 区域 | 内容 |
|------|------|
| 基本信息 | 姓名、手机、邮箱、应聘岗位、来源、标签、当前状态 |
| 简历 | 查看/下载候选人简历附件 |
| 面试安排 | 所有面试轮次的时间、面试官、地点等信息 |
| 面试评价 | 各轮面试的评分和评语 |
| 时间线 | 候选人的状态变更历史 |

> **提示**：面试前请提前查看候选人简历和基本信息，做好面试准备。

---

## 五、查看面试日程

路径：顶部导航 → **日程**

日程页面展示您即将进行的面试安排：
- 按日期分组显示
- 每条记录包含：候选人姓名、面试轮次、时间、地点/会议链接
- 点击候选人姓名可跳转至详情页

### 飞书日历同步
如果 HR 在安排面试时启用了飞书日历同步，您也可以在 **飞书日历** 中直接查看面试安排。

---

## 六、提交面试评价

面试评价是面试官最核心的操作。有两种方式提交：

### 6.1 方式一：系统内提交

1. 登录系统
2. 进入候选人详情页
3. 找到对应的面试轮次
4. 点击 **"提交评价"** 按钮
5. 填写评价表单

### 6.2 方式二：通过评价链接提交（无需登录）

HR 安排面试后，系统会通过飞书消息发送 **面试评价链接** 给您。

1. 点击飞书消息中的评价链接
2. 直接打开评价页面（无需登录系统）
3. 填写评价表单并提交

> 这种方式特别方便，即使您不常使用招聘系统也能快速完成面评。

### 6.3 评价表单字段

| 字段 | 说明 |
|------|------|
| 评分等级 | 从以下选项中选择一个：**S**（优秀）/ **A**（良好）/ **B+**（较好）/ **B**（一般）/ **B-**（较差）/ **C**（不合格）|
| 评价内容 | 对候选人的综合评价，包括面试表现、能力评估等 |
| 优势 | 候选人的突出优点 |
| 不足 | 候选人需要改进的地方 |
| 是否推荐 | 是否建议进入下一轮 |

### 6.4 评分标准参考

| 等级 | 含义 | 建议操作 |
|------|------|---------|
| S | 能力远超岗位要求，强烈推荐 | 推进到下一轮 |
| A | 能力超出岗位要求，推荐 | 推进到下一轮 |
| B+ | 能力达标，可以考虑 | 视情况推进 |
| B | 能力基本达标，有所顾虑 | 需要讨论 |
| B- | 能力略有不足 | 一般不推荐 |
| C | 明显不符合要求 | 建议淘汰 |

### 6.5 评价提交后的流程
- 评价结果会记录在候选人的时间线中
- HR 会收到飞书通知，得知面评已提交
- HR 根据评价结果决定候选人是否进入下一轮

---

## 七、看板视图

路径：顶部导航 → **看板**

看板以可视化卡片的形式展示候选人的招聘进展：

### 7.1 两种视图
- **流水线视图**：将流程聚合为 6 大阶段（简历筛选、面试中、待发Offer、Offer已发、已入职、淘汰）
- **状态视图**：展示全部 16 个细分状态

### 7.2 查看方式
- 每张卡片代表一位候选人，显示姓名、岗位等信息
- 点击卡片可进入候选人详情页
- 可通过顶部下拉框按岗位筛选

> **注意**：成员角色只能查看与自己相关的岗位候选人，无法拖拽操作卡片。

---

## 八、飞书通知说明

系统会通过飞书自动发送以下通知：

| 通知类型 | 触发时机 | 说明 |
|---------|---------|------|
| 面试安排 | HR 为您安排面试时 | 包含候选人信息、面试时间、地点、评价链接 |
| 面试提醒 | 面试即将开始前 | 通过飞书日历提醒 |
| 评价催促 | 面试结束后未提交面评 | 提醒您及时提交评价 |

> **重要**：收到面试安排通知后，请及时查看候选人资料做好准备；面试结束后请尽快提交评价。

---

## 九、常见操作流程

### 9.1 标准面试流程

```
收到飞书面试通知
    ↓
查看候选人资料（点击通知链接或登录系统）
    ↓
查看候选人简历
    ↓
按时参加面试
    ↓
面试结束后提交评价（系统内或通过评价链接）
    ↓
完成 ✅
```

### 9.2 快速面评流程（推荐）

```
1. 点击飞书消息中的面试评价链接
2. 直接进入评价页面
3. 选择评分等级（S/A/B+/B/B-/C）
4. 填写评价内容、优势和不足
5. 点击提交
6. 完成 ✅
```

## 十、常见问题

### Q1：我看不到某些候选人？
**A**：成员角色只能查看与自己分配岗位相关的候选人。如需查看其他岗位候选人，请联系 HR 管理员调整权限或分配。

### Q2：面试评价链接打不开？
**A**：评价链接包含一次性 Token，请确保：
- 使用链接中的完整 URL
- 链接未过期
- 如仍有问题，可登录系统在候选人详情页中提交评价

### Q3：我可以修改已提交的面试评价吗？
**A**：提交后的评价会记录在时间线中。如需修改，请联系 HR 管理员处理。

### Q4：如何查看我今天有哪些面试？
**A**：两种方式：
1. 登录系统首页，查看"今日面试"板块
2. 查看飞书日历中的面试日程

### Q5：面试时间有冲突怎么办？
**A**：请及时联系安排面试的 HR，协商调整面试时间。

### Q6：我可以自己安排面试吗？
**A**：不可以。面试安排由 HR/管理员负责。如果您认为某位候选人需要加面，请与 HR 沟通。
